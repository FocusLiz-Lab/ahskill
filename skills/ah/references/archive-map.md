# Alex Hormozi IMA Topic Map

Use this map only for IMA query planning.

## Expected IMA Source Types

- Book PDFs or outlines: `$100M Offers`, `$100M Leads`, `$100M Money Models`, playbooks, and handbooks.
- YouTube transcripts: examples, tactical explanations, live Q&A, long-form thinking.
- X/Twitter collections: concise principles, aphorisms, short-form ideas.
- Topic folders or tags: use visible names as source labels in answers.

## Topic Routes

1. 创业起步与商业模式: starting a business, choosing markets, business model fundamentals.
2. 销售成交与高客单价: sales, closing, objections, high-ticket offers, scripts.
3. 营销获客与品牌增长: marketing, ads, content, brand, lead generation, hooks.
4. 定价利润与财富积累: pricing, margins, profit, money, wealth building.
5. 业务增长与规模化运营: hiring, systems, operations, scale, LTV, retention.
6. 创业者心态与人生策略: mindset, discipline, learning, ambition, consistency, life strategy.
7. 案例拆解与实战诊断: case studies, stranger business builds, interviews, Q&A, teardown sessions.
