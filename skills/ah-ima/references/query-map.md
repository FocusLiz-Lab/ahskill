# Alex Hormozi IMA Query Map

Use this map to design IMA searches.

## Source Types To Prefer

- `$100M Offers`: canonical offer, value equation, risk reversal, bonuses, urgency, scarcity.
- `$100M Leads`: lead magnets, channels, core four, ads, content, outreach, referrals.
- `$100M Money Models`: monetization, profit, cash flow, wealth, business model design.
- Playbooks and handbooks: applied checklists for ads, closing, hooks, pricing, retention, LTV.
- YouTube transcripts: examples, tactical explanations, long-form Q&A and teardowns.
- X/Twitter collections: concise principles and recent short-form ideas.
